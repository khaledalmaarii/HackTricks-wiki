Se hai un pcap contenente la comunicazione via USB di una tastiera come quella seguente:

![](<../../../.gitbook/assets/image (962).png>)

Puoi utilizzare lo strumento [**ctf-usb-keyboard-parser**](https://github.com/carlospolop-forks/ctf-usb-keyboard-parser) per ottenere ci√≤ che √® stato scritto nella comunicazione:
```bash
tshark -r ./usb.pcap -Y 'usb.capdata && usb.data_len == 8' -T fields -e usb.capdata | sed 's/../:&/g2' > keystrokes.txt
python3 usbkeyboard.py ./keystrokes.txt
```
Puoi leggere ulteriori informazioni e trovare alcuni script su come analizzare questo in:
