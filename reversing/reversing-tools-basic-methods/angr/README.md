* Se voc√™ gostaria que `argc` fosse simb√≥lico, voc√™ pode passar um bitvector simb√≥lico como `argc` para os construtores `entry_state` e `full_init_state`. * Para usar o estado de chamada, voc√™ deve cham√°-lo com `.call_state(addr, arg1, arg2, ...)`, onde `addr` √© o endere√ßo da fun√ß√£o que voc√™ quer chamar e `argN` √© o N-√©simo argumento para essa fun√ß√£o, seja como um inteiro python, string ou array, ou um bitvector.
