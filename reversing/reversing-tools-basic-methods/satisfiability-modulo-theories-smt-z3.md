# Osnovne operacije

## Booleans/And/Or/Not
```python
#pip3 install z3-solver
from z3 import *
s = Solver() #The solver will be given the conditions

x = Bool("x") #Declare the symbos x, y and z
y = Bool("y")
z = Bool("z")

# (x or y or !z) and y
s.add(And(Or(x,y,Not(z)),y))
s.check() #If response is "sat" then the model is satifable, if "unsat" something is wrong
print(s.model()) #Print valid values to satisfy the model
```

## Ispisivanje modela

To print the model, you can use the `model` object obtained from the solver. The `model` object contains the assignments for each variable in the formula. 

Za ispisivanje modela, mo쬰te koristiti `model` objekat dobijen od re코ava캜a. `Model` objekat sadr쬴 dodeljene vrednosti za svaku promenljivu u formuli.
```python

x, y, z = Reals('x y z')
s = Solver()
s.add(x > 1, y > 1, x + y > 3, z - x < 10)
s.check()

m = s.model()
print ("x = %s" % m[x])
for d in m.decls():
print("%s = %s" % (d.name(), m[d]))
```

## Potpisani/Nepotpisani brojevi

Z3 pru쬬 posebne potpisane verzije aritmeti캜kih operacija gde je va쬹o da li se **bit-vektor tretira kao potpisan ili nepotpisan**. U Z3Py, operatori **<, <=, >, >=, /, % i >>** odgovaraju **potpisanim** verzijama. Odgovaraju캖i **nepotpisani** operatori su **ULT, ULE, UGT, UGE, UDiv, URem i LShR.**
```python
from z3 import *

# Create to bit-vectors of size 32
x, y = BitVecs('x y', 32)
solve(x + y == 2, x > 0, y > 0)

# Bit-wise operators
# & bit-wise and
# | bit-wise or
# ~ bit-wise not
solve(x & y == ~y)
solve(x < 0)

# using unsigned version of <
solve(ULT(x, 0))
```
## Funkcije

**Tuma캜ene funkcije** kao 코to je aritmetika, gde **funkcija +** ima **fiksnu standardnu interpretaciju** (sabira dva broja). **Neterpretirane funkcije** i konstante su **maksimalno fleksibilne**; dozvoljavaju **bilo koju interpretaciju** koja je **konzistentna** sa **ograni캜enjima** nad funkcijom ili konstantom.

Primer: dva puta primenjena funkcija f na x rezultuje ponovo x, ali jednom primenjena funkcija f na x je razli캜ita od x.
```python

m = s.model()
```

```python
    solver = Solver()

    # Definisanje promenljivih za svako polje u mre쬴
    cells = [[Int(f"cell_{i}_{j}") for j in range(9)] for i in range(9)]

    # Dodavanje ograni캜enja za vrednosti polja
    for i in range(9):
        for j in range(9):
            # Vrednosti polja moraju biti izme캠u 1 i 9
            solver.add(And(cells[i][j] >= 1, cells[i][j] <= 9))

    # Dodavanje ograni캜enja za redove
    for i in range(9):
        solver.add(Distinct(cells[i]))

    # Dodavanje ograni캜enja za kolone
    for j in range(9):
        solver.add(Distinct([cells[i][j] for i in range(9)]))

    # Dodavanje ograni캜enja za kvadrante
```
