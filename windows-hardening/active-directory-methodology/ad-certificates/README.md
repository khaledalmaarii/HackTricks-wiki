4. **–°–ª—É–∂–±–∞ —Ä–µ—î—Å—Ç—Ä–∞—Ü—ñ—ó —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç—ñ–≤** (CES), —É –ø–æ—î–¥–Ω–∞–Ω–Ω—ñ –∑ —Å–ª—É–∂–±–æ—é –ø–æ–ª—ñ—Ç–∏–∫–∏ —Ä–µ—î—Å—Ç—Ä–∞—Ü—ñ—ó —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç—ñ–≤ (CEP).
5. **–°–ª—É–∂–±–∞ —Ä–µ—î—Å—Ç—Ä–∞—Ü—ñ—ó –º–µ—Ä–µ–∂–µ–≤–∏—Ö –ø—Ä–∏—Å—Ç—Ä–æ—ó–≤** (NDES) –¥–ª—è –º–µ—Ä–µ–∂–µ–≤–∏—Ö –ø—Ä–∏—Å—Ç—Ä–æ—ó–≤, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ –ø—Ä–æ—Å—Ç–∏–π –ø—Ä–æ—Ç–æ–∫–æ–ª —Ä–µ—î—Å—Ç—Ä–∞—Ü—ñ—ó —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç—ñ–≤ (SCEP).

–ö–æ—Ä–∏—Å—Ç—É–≤–∞—á—ñ Windows —Ç–∞–∫–æ–∂ –º–æ–∂—É—Ç—å –∑–∞–ø–∏—Ç—É–≤–∞—Ç–∏ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∏ —á–µ—Ä–µ–∑ GUI (`certmgr.msc` –∞–±–æ `certlm.msc`) –∞–±–æ –∫–æ–º–∞–Ω–¥–Ω—ñ —ñ–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–∏ (`certreq.exe` –∞–±–æ –∫–æ–º–∞–Ω–¥—É PowerShell `Get-Certificate`).

## –°–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–Ω–∞ –ê—É—Ç–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—è

Active Directory (AD) –ø—ñ–¥—Ç—Ä–∏–º—É—î —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–Ω—É –∞—É—Ç–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—é, –≤ –æ—Å–Ω–æ–≤–Ω–æ–º—É –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ **Kerberos** —Ç–∞ **Secure Channel (Schannel)** –ø—Ä–æ—Ç–æ–∫–æ–ª–∏.

### –ü—Ä–æ—Ü–µ—Å –ê—É—Ç–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—ó Kerberos

–£ –ø—Ä–æ—Ü–µ—Å—ñ –∞—É—Ç–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—ó Kerberos –∑–∞–ø–∏—Ç –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ –Ω–∞ –æ—Ç—Ä–∏–º–∞–Ω–Ω—è –∫–≤–∏—Ç–∫–∞ –Ω–∞ –æ—Ç—Ä–∏–º–∞–Ω–Ω—è –∫–≤–∏—Ç–∫–∞ (TGT) –ø—ñ–¥–ø–∏—Å—É—î—Ç—å—Å—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é **–ø—Ä–∏–≤–∞—Ç–Ω–æ–≥–æ –∫–ª—é—á–∞** —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞. –¶–µ–π –∑–∞–ø–∏—Ç –ø—Ä–æ—Ö–æ–¥–∏—Ç—å –∫—ñ–ª—å–∫–∞ –ø–µ—Ä–µ–≤—ñ—Ä–æ–∫ –¥–æ–º–µ–Ω–Ω–∏–º –∫–æ–Ω—Ç—Ä–æ–ª–µ—Ä–æ–º, –≤–∫–ª—é—á–∞—é—á–∏ **–¥—ñ–π—Å–Ω—ñ—Å—Ç—å** —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞, **—à–ª—è—Ö** —Ç–∞ **—Å—Ç–∞—Ç—É—Å –≤—ñ–¥–∫–ª–∏–∫–∞–Ω–Ω—è**. –ü–µ—Ä–µ–≤—ñ—Ä–∫–∏ —Ç–∞–∫–æ–∂ –≤–∫–ª—é—á–∞—é—Ç—å –ø—ñ–¥—Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è —Ç–æ–≥–æ, —â–æ —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç –ø–æ—Ö–æ–¥–∏—Ç—å –∑ –Ω–∞–¥—ñ–π–Ω–æ–≥–æ –¥–∂–µ—Ä–µ–ª–∞, —Ç–∞ –ø—ñ–¥—Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è –Ω–∞—è–≤–Ω–æ—Å—Ç—ñ –≤–∏–¥–∞–≤—Ü—è –≤ **NTAUTH —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–Ω–æ–º—É —Å—Ö–æ–≤–∏—â—ñ**. –£—Å–ø—ñ—à–Ω—ñ –ø–µ—Ä–µ–≤—ñ—Ä–∫–∏ –ø—Ä–∏–∑–≤–æ–¥—è—Ç—å –¥–æ –≤–∏–¥–∞—á—ñ TGT. –û–±'—î–∫—Ç **`NTAuthCertificates`** –≤ AD, —Ä–æ–∑—Ç–∞—à–æ–≤–∞–Ω–∏–π –∑–∞:

### Secure Channel (Schannel) –ê—É—Ç–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ü—ñ—è

Schannel –∑–∞–±–µ–∑–ø–µ—á—É—î –±–µ–∑–ø–µ—á–Ω—ñ TLS/SSL –∑'—î–¥–Ω–∞–Ω–Ω—è, –¥–µ –ø—ñ–¥ —á–∞—Å —Ä—É–∫–æ—Å—Ç–∏—Å–∫–∞–Ω–Ω—è –∫–ª—ñ—î–Ω—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—î —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç, —è–∫–∏–π, —è–∫—â–æ —É—Å–ø—ñ—à–Ω–æ –ø–µ—Ä–µ–≤—ñ—Ä–µ–Ω–∏–π, –Ω–∞–¥–∞—î –¥–æ—Å—Ç—É–ø. –í—ñ–¥–æ–±—Ä–∞–∂–µ–Ω–Ω—è —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞ –Ω–∞ –æ–±–ª—ñ–∫–æ–≤–∏–π –∑–∞–ø–∏—Å AD –º–æ–∂–µ –≤–∫–ª—é—á–∞—Ç–∏ —Ñ—É–Ω–∫—Ü—ñ—é Kerberos **S4U2Self** –∞–±–æ **Subject Alternative Name (SAN)** —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–∞, —Å–µ—Ä–µ–¥ —ñ–Ω—à–∏—Ö –º–µ—Ç–æ–¥—ñ–≤.

### –ü–µ—Ä–µ—Ä–∞—Ö—É–≤–∞–Ω–Ω—è —Å–µ—Ä—Ç–∏—Ñ—ñ–∫–∞—Ç–Ω–∏—Ö —Å–ª—É–∂–± AD

–ö–æ–º–∞–Ω–¥–∏ –¥–ª—è –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è —Ü–∏—Ö —ñ–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç—ñ–≤ –≤–∫–ª—é—á–∞—é—Ç—å:
