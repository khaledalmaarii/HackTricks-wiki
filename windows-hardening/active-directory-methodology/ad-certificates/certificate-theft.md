## NTLM Credential Theft via PKINIT ‚Äì THEFT5

–î–∞–Ω–∏–π –∫–æ–Ω—Ç–µ–Ω—Ç –ø–æ—è—Å–Ω—é—î –º–µ—Ç–æ–¥ –∫—Ä–∞–¥—ñ–∂–∫–∏ –æ–±–ª—ñ–∫–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö NTLM —á–µ—Ä–µ–∑ PKINIT, –∑–æ–∫—Ä–µ–º–∞ —á–µ—Ä–µ–∑ –º–µ—Ç–æ–¥ –∫—Ä–∞–¥—ñ–∂–∫–∏, –ø–æ–∑–Ω–∞—á–µ–Ω–∏–π —è–∫ THEFT5. –û—Å—å –ø–æ–≤—Ç–æ—Ä–Ω–µ –ø–æ—è—Å–Ω–µ–Ω–Ω—è –≤ –ø–∞—Å–∏–≤–Ω–æ–º—É –≥–æ–ª–æ—Å—ñ, –∑ –∞–Ω–æ–Ω—ñ–º—ñ–∑–∞—Ü—ñ—î—é —Ç–∞ —É–∑–∞–≥–∞–ª—å–Ω–µ–Ω–Ω—è–º, –¥–µ —Ü–µ –¥–æ—Ä–µ—á–Ω–æ:
